Module D

Structural Model Assessment

The "Inner Model". We test the relationships (hypotheses) between your constructs and evaluate the model's explanatory power.

1. Path Coefficients & Hypotheses

This answers your research questions. Is there a relationship? Is it positive or negative?

The Beta Value (β)

The Path Coefficient (Standardized Beta) ranges from -1 to +1.

  • Positive (+): As Variable A increases, Variable B increases.
  • Negative (-): As Variable A increases, Variable B decreases.
  • Value Magnitude: Closer to 1 means a stronger relationship.
Example
β = 0.45 means a moderate positive relationship.
0.45

Why do we need Bootstrapping?

The PLS-SEM algorithm calculates the Path Coefficients, but it doesn't tell us if they are "Real" or just "Luck".

We use Bootstrapping to check stability.

Process: SmartPLS creates 5,000 (or more) random sub-samples from your data. It calculates the path for each sub-sample to see if the result is consistent.

Software Steps:

  1. Calculate: Bootstrapping.
  2. Sign: "Sign Changes" set to "No sign changes" (usually).
  3. Subsamples: 5,000.
  4. Run.

The T-Value & P-Value

Look at the "T Statistics" column in the Bootstrapping results.

Significant (p < 0.05)
Standard for most journals.
|T| ≥ 1.96
Significant (p < 0.10)
Sometimes accepted in exploratory studies.
|T| ≥ 1.645

Hypothesis Decision

Supported: T-value > 1.96 AND the Sign matches your hypothesis (e.g., you predicted positive, and result is +).

Not Supported: T-value < 1.96 (insignificant) OR Sign is opposite.

2. Explanatory Power (R² & f²)

How well does your model explain the data?

Coefficient of Determination (R²)

R² measures the variance in the Dependent Variable explained by the Independent Variables.

It is an "In-Sample" measure (how well the model explains the data we currently have).

Thresholds

Weak
0.25
Moderate
0.50
Substantial
0.75

Effect Size (f²)

Question: "Even if the path is significant, does it actually matter?"

f² calculates the change in R² when a specific predictor is removed from the model.

Remove Construct A -> R² drops from 0.60 to 0.55 -> Effect is calculated.

Thresholds

Small Effect
f² ≥ 0.02
Medium Effect
f² ≥ 0.15
Large Effect
f² ≥ 0.35

Note: f² values below 0.02 imply the predictor has no substantive impact, even if significant.

3. Predictive Relevance (Q²)

Can the model predict data it hasn't seen?

Stone-Geisser Q² (Blindfolding)

R² is "In-Sample". Q² is "Out-of-Sample".

SmartPLS temporarily hides parts of the data (omission distance d), uses the model to predict the hidden values, and checks the error.

  • Calculation: 1 - (Sum of Squared Errors / Sum of Squares Prediction)

The Golden Rule of Q²

Predictive Relevance
The model has predictive power.
Q² > 0

If Q² < 0, your model lacks predictive relevance. You might need to rethink your model or include better predictors.

Effect Sizes: Small (0.02), Medium (0.15), Large (0.35).

4. Model Fit Indices (SRMR)

While PLS-SEM focuses on prediction, reviewers often ask for "Model Fit".

Standardized Root Mean Square Residual (SRMR)

SRMR is the difference between the observed correlations (your data) and the implied correlations (your model).

Good Fit
Henseler et al. (2014) threshold.
< 0.08
Acceptable Fit
Hu & Bentler (1999) threshold.
< 0.10
