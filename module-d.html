Module D

Structural Model Assessment

The structural (inner) model evaluates the theoretical relationships between constructs. This is where hypotheses are tested and theory is confirmed or rejected.

0. Correct Structural Model Workflow

  1. Confirm measurement model validity (Module C)
  2. Check collinearity (Inner VIF)
  3. Assess path coefficients & significance (Bootstrapping)
  4. Evaluate explanatory power (R², f²)
  5. Evaluate predictive relevance (Q²)
  6. Report model fit (SRMR – optional but recommended)
Critical Rule
Never interpret paths if measurement validity is not established.

1. Collinearity Assessment (Inner VIF)

Collinearity occurs when predictor constructs are highly correlated, inflating standard errors and destabilizing path estimates.

Inner VIF Threshold
Collinearity is not a concern
< 3.3

Check VIF values for each endogenous construct in SmartPLS before interpreting path coefficients.

2. Path Coefficients & Hypothesis Testing

Path Coefficient (β)

The standardized path coefficient indicates the direction and strength of the relationship between two constructs.

  • β > 0: Positive relationship
  • β < 0: Negative relationship
  • |β| closer to 1: Stronger effect
β = 0.45 → A one standard deviation increase in X increases Y by 0.45 standard deviations.

Why Bootstrapping?

PLS-SEM does not assume normality. Bootstrapping provides empirical sampling distributions to assess significance.

  1. Calculate → Bootstrapping
  2. Subsamples: 5,000 (minimum)
  3. Sign changes: No sign changes

Significance Thresholds

p < 0.05
Hypothesis supported
|t| ≥ 1.96
p < 0.10
Exploratory acceptance
|t| ≥ 1.645

Both statistical significance and sign consistency with theory are required.

3. Explanatory Power (R² & f²)

Coefficient of Determination (R²)

R² represents the proportion of variance in an endogenous construct explained by its predictors.

Weak
0.25
Moderate
0.50
Substantial
0.75

Effect Size (f²)

f² evaluates the contribution of each predictor construct to an endogenous variable.

  • Small: f² ≥ 0.02
  • Medium: f² ≥ 0.15
  • Large: f² ≥ 0.35

A significant path with negligible f² may lack practical importance.

4. Predictive Relevance (Q²)

Q² assesses out-of-sample predictive capability using the blindfolding procedure.

Predictive Relevance
Model predicts unseen data
Q² > 0

Effect size interpretation: 0.02 (small), 0.15 (medium), 0.35 (large).

5. Model Fit (SRMR)

Although prediction-oriented, PLS-SEM studies often report SRMR as a global fit index.

Good Fit
Henseler et al. (2014)
< 0.08
Acceptable
Hu & Bentler (1999)
< 0.10
